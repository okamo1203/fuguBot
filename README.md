# fuguBot
Bluemix上で動くWatson APIを使ったSlack Botの作成  
ふぐ(bot)の育成方法の記録  

### step1 ふぐの仕入れと育成準備
1. ふぐの仕入れ  
以下のページでSlack Botを作成する  
https://my.slack.com/services/new/bot  
　
APIトークンをメモしよう（大事）  
気がついたらここで ふぐ のbotを作ってた。。。  

2. 育成準備（ふぐを水槽にいれます）  
Slackに自分のIDでログインし、botを登録したいチャネルを選んだ後、”/invite @[botの名前]”と入力します。  

これで準備OK！
